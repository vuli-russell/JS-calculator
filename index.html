Calculator

More Info

A calculator built in JS, with styling added via CSS.

The calculator can do any normal maths, accoridng to BODMAS, and can store 2 variables, x and y, to be used with calculations

The calculator works with clicks or keystrokes.

Rotation is controlled via w,a,s,d,q and e for +/- in 3 axis of rotation, or very sketchily via mouse drag

Theres basically no error checking, so if something goes wrong, clear and start again, and BE MORE CAREFUL!

There is a 100% chance that the device you are accesing this on has a better calculator pre installed - please use that instead.
