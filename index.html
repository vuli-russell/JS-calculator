Calculator

More Info

A calculator built in JS, with roation added via CSS

The calculator can do any normal maths, accoridng to BODMAS, and can store 2 variables, x and y, to be used with calculations

The calculator works with clicks or keystrokes.

Rotation is controlled via w,a,s,d,q and e for the 6 axis of rotation.

Theres basically no error checking, so if something goes wrong, clear and start again, and BE MORE CAREFUL!
